# GREEN-API Test Client

Интерфейс для тестирования методов GREEN-API:
- getSettings
- getStateInstance  
- sendMessage
- sendFileByUrl


## 🚀 Использование
1. Создайте инстанс в [GREEN-API](https://green-api.com)
2. Отсканируйте QR-код для авторизации
3. Введите `idInstance` и `apiTokenInstance` на странице
4. Используйте кнопки для вызова методов

## 📋 Требования
- Современный браузер с поддержкой Fetch API
- Активный инстанс GREEN-API

## ⚠️ Важно
- Этот клиент выполняет запросы напрямую из браузера (CORS разрешён GREEN-API)
